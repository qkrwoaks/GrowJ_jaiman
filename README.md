# 짜이만 키우기

2021.11 ~ 2022.01

---

**짜이만 키우기**는 **거지키우기**를 패러디한 **방치형 클리커 2D 게임**입니다.<br>
저희는 가상 인물 짜이만을 만들어, 거지 키우기 처럼 게임을 진행하되, 저희 학교에서 게임 개발로 성장하는 스토리를 담고자 하였습니다.
